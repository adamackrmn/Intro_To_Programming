Variables & Expressions

Every application you will write, no matter how big or how small, will always use these two concepts. We will be learning how to create/initialize, assign and update variables.

Variables

Variables are used to store values of various types.

Values could come from :
  • The user of the application
  • A database
  • A file / web-service
  • Something that you derived within the application
There are two types of variables, Primitive, sometimes referred to as Simple, and Complex

Primitive Data Types

Type .NET Type Size
bool System.Boolean 0.125-bytes (1-bit)
char System.Char 2-bytes (16-bits)
int System.Int32 4-bytes (32-bits)
double System.Double 8-bytes (64-bits)

8 bits = 1 byte
